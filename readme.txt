Ce plugin wordpress vous permet d'envoyer des SMS à vos membres ou aux numéros de votre choix depuis votre admin Wordpress

== Description ==

Ce plugin Wordpress vous permet d'envoyer des SMS à vos membres (ou à des numéros saisis manuellement) directement depuis l'administration de votre site Wordpress.
Il vous permet également de recevoir une notification par SMS lorsqu'un nouveau commentaire ou post est créé sur votre Blog.
Vous aurez accès, directement depuis votre console d'administration, à l'historique des envois SMS de votre compte SMS ENVOI ainsi qu'aux accusés de réception SMS.

**L'utilisation de ce plugin nécessite la création d'un compte gratuit d'envoi de SMS sur [SMSenvoi.com](http://www.smsenvoi.com/membres/inscription/ "Inscription SMSENVOI") ainsi qu'un nombre de crédits SMS suffisant.**

5 SMS Tests **offerts** sur simple demande par e-mail à contact@smsenvoi.com


**1 crédit = 1 SMS,  Rien de plus simple !**

**A partir de 0,03 € par SMS**



* Différentes gammes adaptées à vos besoins :
1. SMS Low Cost
2. SMS Standard
3. SMS Premium
4. SMS Long
5. SMS Stop


SMS ENVOI est le partenaire de référence de nombreuses enseignes de la grande distribution. Aucun volume minimum d'envoi n'est requis.

== Installation ==

L'installation du plugin peut se faire de 2 fa&ccedil;ons :


* M&eacute;thode automatisée :

1. Recherchez l'extension "SMS ENVOI" depuis le menu "Extensions" de Wordpress
2. Installez et activez le plugin
3. Paramétrez le plugin avec vos informations de compte SMS ENVOI dans la section "Réglages" de votre interface d'administration Wordpress


* M&eacute;thode manuelle :

1. [T&eacute;l&eacute;chargez le plugin](http://www.smsenvoi.com/site/webroot/API/wordpress-smsenvoi-1.0.0.zip "Télécharger") puis décompressez l'archive
2. Uploadez le dossier `smsenvoi` dans le r&eacute;pertoire `/wp-content/plugins/` de votre site wordpress
3. Activez le plugin dans le menu `Extensions` de Wordpress
4. Paramétrez le plugin avec vos informations de compte SMS ENVOI dans la section "Réglages" de votre interface d'administration Wordpress

**L'utilisation de ce plugin nécessite la création d'un compte gratuit sur [SMSenvoi.com](http://www.smsenvoi.com/membres/inscription/ "Inscription SMSENVOI") ainsi qu'un nombre de crédits SMS suffisant.**

== Informations techniques ==

Le plugin ajoute deux nouveaux champs aux comptes utilisateurs de votre site Wordpress : Un champ contenant leur numéro de téléphone ainsi qu'un champ optin leur permettant d'accepter ou non de recevoir un SMS
Seuls les utilisateurs ayant accepté de recevoir un SMS en recevront.
