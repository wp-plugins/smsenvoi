This plugin allows you to send SMS to your members, and to set up alerts for the admin. It extends member profile to add 2 new fields. One containing the user cellphone, the other one is an optin field. Only user checking this optin will receive SMS.
You need to create a free account on http://www.smsenvoi.com ( [http://www.smsenvoi.com/membres/inscription/] ) first and then to buy credits in order to be able to send SMS.

FR: Ce plugin wordpress vous permet d'envoyer des SMS à vos membres directement depuis votre interface d'administration

== Description ==

Ce plugin Wordpress vous permet d'envoyer des SMS à vos membres (ou à des numéros saisis manuellement) directement depuis l'administration de votre site Wordpress.
Il vous permet également de recevoir une notification par SMS lorsqu'un nouveau commentaire ou post est créé sur votre Blog.
Vous aurez accès, directement depuis votre console d'administration, à l'historique des envois de votre compte SMS ENVOI ainsi qu'aux accusés de réception.

L'utilisation de ce plugin nécessite la création d'un compte gratuit sur http://www.smsenvoi.com ( [http://www.smsenvoi.com/membres/inscription/] ) ainsi qu'un nombre de crédits SMS suffisant.


== Installation ==

L'installation du plugin peut se faire de 2 fa&ccedil;ons :


* M&eacute;thode automatisée :

1. Recherchez l'extension "SMS ENVOI" depuis le menu "Extensions" de Wordpress
2. Installez et activez le plugin
3. Paramétrez le plugin avec vos informations de compte SMS ENVOI dans la section "Réglages" de votre interface d'administration Wordpress


* M&eacute;thode manuelle :

1. T&eacute;l&eacute;chargez le plugin [http://www.smsenvoi.com/site/webroot/API/wordpress-smsenvoi-1.0.0.zip] puis décompressez l'archive
2. Uploadez le dossier `Wordpress-send-SMS` dans le r&eacute;pertoire `/wp-content/plugins/` de votre site wordpress
3. Activez le plugin dans le menu `Extensions` de Wordpress
4. Paramétrez le plugin avec vos informations de compte SMS ENVOI dans la section "Réglages" de votre interface d'administration Wordpress
